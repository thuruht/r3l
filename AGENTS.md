# AGENTS.md — R3L:F

Cloudflare-native social network (ephemeral file sharing + D3 force graph). Hono worker + React 18 SPA.

## Commands

```bash
npm run dev              # wrangler dev (serves both worker & Vite frontend)
npm run build:client     # Vite build (root=client/, output=client/dist/)
npm run deploy           # build:client then wrangler deploy --minify
npx wrangler d1 migrations apply relf-db --local   # apply D1 migrations locally
npx wrangler d1 migrations apply relf-db --remote  # apply to prod

# Tests (3 configs — must specify --config)
npx vitest run --config vitest.config.ts             # backend D1 integration tests (src/**/*.test.ts, Cloudflare pool)
npx vitest run --config vitest.unit.config.ts         # unit tests (src/__tests__/unit/**/*.test.ts)
npx vitest run --config vitest.integration.config.ts  # integration tests (forks pool)
```

No CI, no linter, no formatter, no typecheck script.

## Architecture

### Two-tier
- **Worker** (`src/index.ts`): Hono router. All API under `/api/*`. Exports `RelfDO`, `DocumentRoom`, `ChatRoom` DO classes.
- **Frontend** (`client/src/`): React 18 + Vite PWA. `@` alias → `client/src/`. Vite root is `client/`.

### Cloudflare Bindings (`wrangler.jsonc`)
| Binding | Type | Purpose |
|---|---|---|
| `DB` | D1 | relational data |
| `BUCKET` | R2 | file content storage |
| `KV` | KV | sessions, rate limiting, drift history (`history:{userId}`) |
| `DO_NAMESPACE` | DO (`RelfDO`) | WebSocket presence + notifications |
| `DOCUMENT_ROOM` | DO (`DocumentRoom`) | Yjs collaborative editing |
| `CHAT_ROOM` | DO (`ChatRoom`) | chat rooms |

### Auth
JWT in HttpOnly cookie `auth_token`. `c.get('user_id')` available on protected routes. Public paths defined in `PUBLIC_PATHS` array at `src/index.ts:109` (login, register, verify-email, forgot/reset-password, `/chat/`, `/discovery/users/search`, `/discovery/users/random`, `/communiques/`). Also `GET /api/users/:id` (numeric).

### Durable Objects
All 3 DOs use **Hibernation API** (`state.acceptWebSocket` + `webSocketMessage`/`webSocketClose`). Session metadata via `ws.serializeAttachment()`. Chat room WebSocket has lighter auth (optional JWT parse, no middleware).

### SPA catch-all
Two layers: wrangler.jsonc `not_found_handling: "single-page-application"` AND `app.all('*')` as the **last** route in `src/index.ts` that delegates to `c.env.ASSETS.fetch()`.

### Cron
Hourly (`0 * * * *`): decrement vitality, send <24h expiry notifications, delete expired files, detect organic resonance.

### Auth/sanitization utilities (`src/utils/security.ts`)
Passwords: PBKDF2-SHA-256, 100k iterations, per-user salt. HTML sanitization uses Cloudflare `HTMLRewriter` on server, `DOMParser` allowlist on client (`client/src/utils/sanitize.ts`).

## Testing

Tests exist in `src/__tests__/` despite CLAUDE.md saying otherwise. Uses `@cloudflare/vitest-pool-workers`. Test setup (`src/__tests__/setup.ts`) applies D1 migrations from auto-generated `src/__tests__/migration-data.ts`.

**Important:** `src/__tests__/migration-data.ts` is auto-generated by `vitest.config.ts` (reads `migrations/*.sql` and writes a `.ts` module). Do not edit manually.

## Frontend conventions

- **CSS variables** from `client/src/styles/design-tokens.css` for all spacing/colors/z-index. No hardcoded values.
- **Icon sizes**: use `ICON_SIZES` from `client/src/constants/iconSizes.ts` for `@tabler/icons-react` (not raw numbers).
- **Terminology register**: UPPERCASE in labels/badges/buttons/headings; lowercase in prose (tooltips, empty states). See `docs/terminology.md`.
- **Component classes**: `.btn-primary`, `.btn-secondary`, `.btn-text`, `.btn-icon`, `.modal-panel`, `.glass-panel`, `.flex-between`, `.flex-center`, `.flex-gap-md` from `client/src/styles/components.css`.
- **Design system docs**: `client/src/DESIGN_SYSTEM.md`.

## Gotchas

- `tsconfig.json`: `strict: false`, `noImplicitAny: false` — types are loose.
- Drift history stored in KV with 30-day TTL, max 50 entries (`src/index.ts:139`).
- Rate limits defined in `src/constants.ts` (drift: 20 req/600s, login: 10 req/600s, etc.).
- DO class names in `src/index.ts` exports must match `wrangler.jsonc` Durable Object bindings.
- File encryption: AES-GCM via `encryptData`/`decryptData` (`src/utils/security.ts`). E2EE messages: RSA-OAEP + AES-GCM, client-side (`client/src/utils/crypto.ts`).
- `.dev.vars` provides dev placeholders for `JWT_SECRET` and `ENCRYPTION_SECRET`.
- Agent handoff doc: `docs/AGENT_HANDOFF.md` has active backlog (threaded comments, bookmarks, undo deletion, media streaming).
