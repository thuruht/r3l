Help & FAQ

Find answers to common questions and learn how to use R3L:F effectively

What is R3L:F?

R3L:F (Relational Ephemeral Filenet) is a platform designed for creating, sharing, and discovering content without algorithmic interference. It emphasizes user control, ephemeral content, and community-driven archiving.

Navigation

The main navigation bar provides access to all R3L:F features:

  • Home: Return to the main landing page
  • Map: Explore content geographically
  • Network: Visualize content and user connections
  • Search: Find content and users
  • Random: Discover random content
  • Messages: Access your conversations
  • User Menu: Access your profile, drawer, settings, and more

Creating an Account

To create an account:

  1. Click "Login" in the top-right corner
  2. Select "Register" on the login page
  3. Choose a username and password
  4. Provide a display name
  5. Complete the registration process

Important: Store your recovery key in a safe place. It's the only way to recover your account if you forget your password.

Understanding Profile vs. Drawer

R3L:F distinguishes between your private profile and public drawer:

  • Profile: Your personal settings page where you can update your information, preferences, and manage your content.
  • Drawer: Your public presence visible to other users, featuring your communiqué and curated content.

Managing Your Profile

To update your profile:

  1. Click your username in the top-right corner
  2. Select "Profile"
  3. Edit your display name, bio, avatar, and other settings
  4. Save your changes

Creating Your Communiqué

Your communiqué is a public statement or message in your drawer:

  1. Go to your drawer (User Menu > Drawer)
  2. Click "Edit Communiqué"
  3. Create your message using the editor
  4. Toggle between edit and preview modes
  5. Save when finished

HTML formatting is supported but sanitized for security.

Avatar Management

To update your avatar:

  1. Go to your Profile
  2. Click on the avatar image or icon
  3. Upload a new image (JPG, PNG) or choose from default options
  4. Save your changes

Your avatar will appear on your profile, drawer, and in conversations.

Creating Content

To create new content:

  1. Click "Upload" in the navigation menu
  2. Select the type of content you want to create
  3. Fill in the title, description, and other details
  4. Set privacy options (public or private)
  5. Upload any associated files
  6. Save your content

Ephemeral Content

By default, content on R3L:F is ephemeral, meaning it has a limited lifespan:

  • Content becomes eligible for expiration after a period of inactivity
  • Community interest (views, saves, votes) can extend content lifespan
  • You can personally archive content to preserve it
  • Community archiving occurs when enough users vote to preserve valuable content

Tagging Content

Tags help organize and discover content:

  1. When creating or editing content, add relevant tags
  2. Separate tags with commas
  3. Use specific, descriptive tags to improve discoverability
  4. Avoid generic or overly broad tags

Adding Geographic Data

You can associate content with geographic locations:

  1. When creating content, select "Add Location"
  2. Choose a location on the map
  3. Alternatively, visit the Map page and click "Add Point"
  4. Fill in location details and link to content if desired

Geographic data helps users discover content through the Map feature.

Connection Types

R3L:F offers different ways to connect with other users:

  • Mutual Connection: Two-way connection requiring acceptance from both parties
  • Follow: One-way connection to track another user's public content
  • Lurk: A more private form of following with enhanced privacy

Finding & Connecting with Users

To connect with other users:

  1. Find users through Search, Map, Network, or content pages
  2. Visit their drawer
  3. Click "Connect" and select the connection type
  4. Add an optional message for mutual connection requests
  5. Wait for acceptance if requesting a mutual connection

Messaging

To send messages to other users:

  1. Click "Messages" in the navigation menu
  2. Select an existing conversation or click "New Message"
  3. Search for and select a user to message
  4. Type your message and press send

You can only message users with whom you have a mutual connection.

Attachments

To include attachments in messages:

  1. In a conversation, click the attachment icon
  2. Select a file to upload
  3. Add a message to accompany the attachment
  4. Send the message

Supported attachment types include images, documents, and other file formats.

Using the Map

The Map feature allows you to explore content geographically:

  • Browse the map to discover content in different locations
  • Filter content by type, date, and other criteria
  • Add your own points of interest
  • Connect geographic points to content

Adding Map Points

To add a point to the map:

  1. Navigate to the Map page
  2. Click "Add Point"
  3. Click on the map to set the location
  4. Fill in the point details (title, description)
  5. Set privacy options
  6. Save the point

Exploring the Network

The Network feature visualizes connections between content and users:

  • See how content is related through tags, authors, and other associations
  • Discover user connections and content networks
  • Zoom, pan, and interact with the visualization
  • Switch between different view modes

Finding Related Content

The Network view helps you discover related content:

  1. Start with a piece of content or a user
  2. Explore the connections that branch out
  3. Click on nodes to view details
  4. Follow connections to discover new content

Lurker in the Mist Mode

"Lurker in the Mist" is a privacy feature that randomizes search results and recommendations:

  • Enables a more serendipitous browsing experience
  • Reduces algorithmic tracking and filter bubbles
  • Adjustable randomness level in your profile settings
  • Helps discover content you might not otherwise encounter

Content Privacy Settings

Control who can see your content:

  • Public: Visible to all users
  • Private: Visible only to you
  • Connections: Visible only to your mutual connections
  • Privacy settings can be set per content item

Account Security

Keeping your account secure:

  • Use a strong, unique password
  • Store your recovery key securely
  • Check active sessions in your profile settings
  • Log out from shared devices

Data & Privacy

R3L:F is designed with privacy in mind:

  • No tracking cookies or surveillance
  • No algorithmic profiling
  • Ephemeral content reduces digital footprint
  • Geographic data is only collected when explicitly shared

Login Issues

If you're having trouble logging in:

  • Check that you're using the correct username and password
  • Clear your browser cache and cookies
  • Try using a different browser
  • Use your recovery key if you've forgotten your password

Content Not Appearing

If content you've created isn't appearing:

  • Check your privacy settings to ensure it's visible
  • Verify that you're logged in
  • Try refreshing the page
  • Check if the content has expired (check your archive)

Missing Notifications

If you're not receiving notifications:

  • Check your notification settings in your profile
  • Make sure you're logged in
  • Try refreshing the page
  • Check your browser permissions for notifications

File Upload Problems

If you're having trouble uploading files:

  • Check that your file is under the size limit (10MB)
  • Ensure the file format is supported
  • Try a different browser
  • Compress large images before uploading
